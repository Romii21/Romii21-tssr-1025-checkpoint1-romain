# addUsers_pseudoCode

## Script de création de compte :

Lancement du script avec des arguments :

argument1=compte1  
argument2=compte2  
argument3=compte3
ect ...

Vérification de présence d'arguments :

  * Les arguments sont présents = message de validation = me script continue
  * Les arguments ne sont pas présents = message d'erreur = le script s'arrete

Vérification de l'éxistannce d'un compte avant de le créer :

  * Le compte à créer :

    * Existe = message d'erreur = le script continu
    * N'existe pas = le script continu

Vérification de la création de compte :

  * Le compte se crée = Message de validation
  * Le compte ne se crée pas = Message d'erreur
